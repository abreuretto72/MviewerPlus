# Política de Privacidade - MviewerPlus

**Última atualização: Dezembro de 2025**

Bem-vindo ao MviewerPlus. Esta Política de Privacidade descreve como coletamos, usamos e protegemos suas informações ao utilizar nosso aplicativo móvel.

## 1. Coleta e Uso de Informações

**O MviewerPlus prioriza a sua privacidade.**

*   **Processamento Local:** A grande maioria das funcionalidades do MviewerPlus (visualização, edição, conversão de arquivos) ocorre inteiramente no seu dispositivo. Não enviamos seus arquivos para servidores externos para processamento básico.
*   **Zero Coleta de Dados Pessoais:** Não exigimos cadastro, login ou fornecimento de informações pessoais (como nome, e-mail ou telefone) para usar as funcionalidades principais do aplicativo.

## 2. Permissões do Dispositivo

Para funcionar corretamente, o aplicativo pode solicitar as seguintes permissões:

*   **Armazenamento (Ler/Escrever):** Necessário para abrir, salvar e modificar os arquivos que você escolher.
*   **Acesso à Internet:** Utilizado verificar atualizações ou acessar links externos (como esta política) e para a funcionalidade de IA (descrita abaixo).

## 3. Funcionalidade de Inteligência Artificial (AI Assistant)

O MviewerPlus oferece um recurso opcional de "Assistente de IA" para analisar documentos.

*   **Provedor Terceiro (Groq):** Este recurso utiliza a API da Groq Inc.
*   **Chave de API Pessoal:** Para usar esta funcionalidade, você deve fornecer sua própria Chave de API (API Key) da Groq. O aplicativo armazena esta chave *apenas localmente* no seu dispositivo (usando armazenamento seguro).
*   **Envio de Dados:** Ao fazer uma pergunta sobre um arquivo, o conteúdo textual desse arquivo e sua pergunta são enviados diretamente dos seu dispositivo para os servidores da Groq para gerar a resposta.
*   **Não-Interferência:** Nós (desenvolvedores do MviewerPlus) não temos acesso, não armazenamos e não processamos o conteúdo dos seus arquivos ou suas chaves de API. A comunicação é direta entre seu App e a Groq.

## 4. Segurança

Empregamos as melhores práticas de desenvolvimento para garantir que seus dados permaneçam seguros no seu dispositivo. No entanto, lembre-se que nenhum método de transmissão pela internet ou armazenamento eletrônico é 100% seguro.

## 5. Serviços de Terceiros e Links

O aplicativo pode conter links para sites externos (como o console da Groq para obter chaves). Não somos responsáveis pelas práticas de privacidade sites de terceiros.

## 6. Alterações nesta Política

Podemos atualizar nossa Política de Privacidade periodicamente. Recomendamos que você revise esta página regularmente para quaisquer alterações.
