Política de Privacidade - MviewerPlus

Última atualização: Dezembro de 2025

Bem-vindo ao MviewerPlus. O processamento de dados do nosso aplicativo é projetado para priorizar a privacidade e o processamento local.

Resumo Rápido para o Google Play:
  • Não criamos contas de usuário (sem login/cadastro).
  • Não vendemos dados de usuários.
  • Não exibimos anúncios (sem SDKs de publicidade).
  • Todo processamento de arquivos é local, exceto quando o usuário opta explicitamente por usar IA.

1. Coleta e Uso de Informações

1.1 Identificadores e Contas

O MviewerPlus não possui sistema de contas. Não exigimos que você faça login, crie uma conta ou forneça e-mail/telefone para usar o aplicativo. Portanto, não retemos dados de cadastro e não há necessidade de procedimentos para "exclusão de conta".

1.2 Processamento de Arquivos e Integridade

Para sua funcionalidade principal (visualizador e editor), o aplicativo acessa os arquivos no armazenamento do seu dispositivo apenas quando você os seleciona.

  • Proteção do Original (Read-Only): O MviewerPlus adota uma política estrita de "Não Destruição". Nunca editamos o arquivo original.
  • Edição Segura: Qualquer modificação cria automaticamente uma cópia independente do arquivo. O arquivo original permanece intocado e preservado no estado exato em que foi aberto.
  • Processamento Local: Todo este processo ocorre 100% local e offline. Nenhum arquivo é enviado para nós.

2. Transmissão de Dados e Terceiros

2.1 Assistente de IA (Opcional)

O aplicativo oferece uma funcionalidade de "Chat com Documentos" usando a API da Groq Inc.

  • Dados Enviados: Apenas quando você interage com o chat, o trecho de texto do arquivo aberto e sua pergunta são enviados para a Groq.
  • Segurança: Toda comunicação é feita via HTTPS (Criptografado).
  • Chave de API: Você utiliza sua própria chave de API. Ela é armazenada apenas no Keychain/SharedPreferences seguro do seu dispositivo e nunca é enviada para nossos servidores.

2.2 Firebase Remote Config

Utilizamos o Google Firebase Remote Config para gerenciar configurações técnicas do aplicativo remotamente (ex: ativar/desativar funcionalidades experimentais). Este serviço pode coletar informações técnicas (como versão do app, sistema operacional e identificadores de instância anonimizados) para funcionamento estritamente técnico.

O Firebase Remote Config não é usado para rastrear o usuário individualmente nem para fins publicitários neste aplicativo.

3. Publicidade e Venda de Dados

  • Sem Anúncios: Este aplicativo não exibe anúncios e não contém SDKs de redes de publicidade (como AdMob ou Facebook Audience Network).
  • Não Venda: Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros.

4. Permissões Solicitadas

  • READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE: Para abrir e salvar arquivos.
  • INTERNET: Para comunicação com a API da IA (se usada) e Firebase Remote Config.
  • USE_BIOMETRIC / USE_FINGERPRINT: Para proteger o acesso ao app (funcionalidade opcional de segurança local).

5. Segurança dos Dados

Adotamos medidas de segurança robustas, incluindo o uso de HTTPS para todas as comunicações de rede e armazenamento seguro para chaves de API. Seus arquivos permanecem sob seu controle no seu dispositivo.
